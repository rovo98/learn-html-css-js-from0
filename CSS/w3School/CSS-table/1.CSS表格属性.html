设置表格框之间的空白

border-spacing属性设置相邻单元格之间的距离(仅适用于“边框分离”模式)

说明

该属性指定分隔边框模型中的单元格边界之间的距离。在指定的两个长度值中，第一个是水平间隔，第二个是垂直间隔。border-collapse必须设置为separate才有效。

默认值: not specified
继承性: yes
版本: CSS2
JavaScript语法: object.style.borderSpacing="15px"

可能的值

值 描述
length length 规定相邻单元格的边框之间的距离。使用px, cm等单位。不允许使用负值。
如果定义一个length参数，那么定义的是水平和垂直间距。
如果定义两个length参数，则第一个是水平间距，第二个是垂直间距。
inherit 规定应该从父元素继承 border-spacing 属性值。

实例

Adams John
Bush George
Carter Thomas
Gates Bill

设置表格标题位置

caption-side属性设置表格标题的位置。

说明

该属性指定了表标题相对于表框的放置位置。表标题显示为好像它是在表之前(之后)的一个块级元素。

默认值: top
继承性: yes
版本: CSS2
JavaScript语法: object.style.captionSide="bottom"

可能的值

值 描述
top 默认值。把表格标题定位在表格之上。
bottom 把表格标题定位在表格之下。
inherit 规定应该从父元素继承 caption-side 属性的值。

实例

将表格标题放置在表格的下面

This is a caption
Adams John
Bush George

注释:如果规定!DOCTYPE, 那么IE 8(以及更高级版本)支持caption-side属性。

隐藏表格空白单元格

定义和用法

empty-cells属性设置是否显示表格中的空白单元格(仅适用于“separate”模式)。

说明

该属性定义了不包含任何内容的表格单元格如何显示。在border-collapse 为 separate 时才有效。

默认值: show
继承性: yes
版本: CSS2
JavaScript语法: object.style.emptyCells="hide"

可能的值

值 描述
hide 不在空单元格周围绘制边框。
show 在空单元格周围绘制边框。默认。
inherit 规定应该从父元素继承 empty-cells 属性的值。

实例

Adams John
Bush

注释:如果规定!DOCTYPE, 那么IE 8(以及更高级版本)支持 empty-cells 属性。

显示表格单元格、行、列的算法规则

定义和用法

tablelayout属性用来显示表格单元格、行、列的算法规则。

固定表格布局

固定表格布局与自动表格布局相比，允许浏览器更快的对表格进行布局。

在固定表格布局中，水平布局仅取决于表格宽度、列宽度、表格边框宽度、单元格间距，而与单元格中的内容无关。

通过使用固定表格布局，客户端在接受第一行后就可以显示表格。

自动表格布局

在自动表格布局中，列的宽度是由列单元格中的没有折行的最宽的内容设定的。

此算法有时会比较慢，这是由于它需要在确定最终布局之前访问表格中的所有内容。

说明

该属性指定了完成表格布局时所用的布局算法。固定表格布局比较快，但是不太灵活，而自动算法比较慢，不过更能反映传统的HTML表格。

默认值: auto
继承性: yes
版本: CSS2
JavaScript语法: object.style.tableLayout="fixed"

可能的值

值 描述
automatic 默认。列宽度有单元格内容设定。
fixed 列宽度由表格宽度和列宽度设定。
inherit 规定应该从父元素继承 table-layout 属性的值。

实例

10000000000000000000000000000000 10000000000 100
10000000000000000000000000000000 10000000000 100
