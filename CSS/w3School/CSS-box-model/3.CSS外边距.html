CSS 外边距 | margin

CSS margin 属性

设置外边距的最简单方法就是使用 margin 属性。

margin 接受任何长度单位，可以是像素、英寸、毫秒或em。

定义和语法

margin 简写在一个声明中设置所有外边距属性。该属性可以有1 到 4 个值。

说明

这个简写属性设置所有外边距的宽度，或者设置各边的外边距宽度。

块级元素的垂直相邻外边距会合并，而行内元素实际上不占上下外边距。行内元素的左右外边距不会合并。浮动元素外边距也不会合并。允许指定负的外边距值，不过使用时要小心。

默认值: 0
继承性: no
版本: CSS1
JavaScript语法: object.style.margin = "10px 5px"

可能的值

值 描述
auto 浏览器计算外边距。
length 规定以具体单位计的外边距值，比如像素、厘米等。默认是 0px。
% 规定基于父元素的宽度的百分比的外边距。
inherit 规定应该从父元素继承外边距。
