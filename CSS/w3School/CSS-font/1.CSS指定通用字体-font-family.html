CSS字体系列

在CSS中，有两种不同类型的字体系列名称:

  • 通用字体系列 - 拥有相似外观的字体系统组合(比如:"Serif" 或 "Monospace")
  • 特定字体系列 - 具体的字体系列(比如: "Times"或"Courier")

除了各种特定的字体系列外，CSS定义了5种通用的字体系列:

  • Serif字体
  • Sans-serif字体
  • Monospace字体
  • Cursive字体
  • Fantasy字体

字体风格

font-style属性最常用于规定斜体文本.
该属性有三个值:

  • normal - 文本正常显示
  • italic - 文本斜体显示
  • oblique - 文本倾斜显示
