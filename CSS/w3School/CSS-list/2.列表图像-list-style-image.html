列表图像 - list-style-image

定义和用法

list-style-image属性使用图像来替换列表项的标记。

说明

这个属性指定一个有序或无序列表标记的图像。图像相对于列表项内容的防止位置可以list-style-position属性来控制。

注释:请始终规定一个"list-style-type"属性以防止图像不可用。

默认值: none
继承性: yes
版本: CSS1
JavaScript语法: object.style.listStyleImage="url('/images/xxx.gif')"

可能的值

值 描述
URL 图像的路径
none 默认。无图形被显示。
inherit 规定应该从父元素继承list-style-image属性的值。

示例：配合list-style-position
